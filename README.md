End-to-end Data Analytics project analyzing digital payment trends using Python, MySQL, SQL, and Streamlit.

Built an ETL pipeline to extract and transform semi-structured JSON data into a structured relational database for efficient querying.

Developed optimized SQL queries to analyze state-wise, district-wise, and yearly transaction performance.

Designed an interactive Streamlit dashboard for real-time data exploration and visualization.

Generated business insights for customer segmentation, regional growth analysis, and insurance adoption trends.

Advanced visualizations were implemented using Matplotlib, Seaborn, and Plotly, including an India choropleth map for geographic insights.

Tech Stack: Python | MySQL | SQL | Pandas | Streamlit | Data Visualization | plotly
